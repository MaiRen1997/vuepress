    vue-router安装使用

    1.首先安装vue-router

    ​ cnpm install vue-router --save

    2.在src下面新建一个router文件夹，然后新建index.js文件

    ​ 在index.js中写入如下代码

    import Vue from "vue";
import VueRouter from 'vue-router'
Vue.use(VueRouter)

const router = new VueRouter({
  routes:[
    {path: '/login', component:() => '../components/login.vue'}
  ]
})

export default router

    3.在main.js中引入router文件夹下暴露出的router

    import Vue from 'vue'
import App from './App.vue'

import router from './router/index'
Vue.config.productionTip = false

new Vue({
  router,
  render: h => h(App),
}).$mount('#app')

    最主要的是import router from './router/index和 在new Vue里引入router,

    4.在App.vue中，使用route-link和route-view

    <template>
  <div id="app">
      <router-link to="/login">登录</router-link>
      <router-view></router-view>
  </div>
</template>

    # vue-router的安装使用

    1. 安装vue-router

      npm install vue-router// vue2 一般安装3.x版本  npm install vue-router@3.5.2

    2. 新建router文件夹,并在里面新建index.js

      import Vue from 'vue'
import VueRouter from 'vue-router'
Vue.use(VueRouter)
const routes =[
  {
    path: '/',
    name:'index',
    component: () => import('../views/pageOne/inde.vue')
  },
  {
    path: '/pageOne',
    name: 'pageOne',
    component: () => import('../views/pageOne/inde.vue')
  },
  {
    path: '/pageTwo',
    name: 'pageTwo',
    component: () => import('../views/pageTwo/index.vue')
  },
]
const router = new VueRouter({
  routes
})
export default router

    3. 在main.js中导入router，并在vue实例中使用路由

      import router from './router/index'
new Vue({
  render: h => h(App),
  router
}).$mount('#app')

    4. 在页面中使用router-link和router-view进行路由跳转
